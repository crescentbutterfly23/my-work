How this page works ▾

Blocks stack down the project page in the order you put them here. Edits save to a local draft as you type — they only go live once you download projects.json, drop it into site/data/, and commit.

Text — the style picker

  • Heading — big Archivo Black caps with a crimson underline. Use to break the page into sections.
  • Lead — oversized statement type. Good for the opening line or a punchy pull-quote.
  • Body — normal readable paragraph.

Image — the width picker

  • 1 / 3 — sits centred at a third of the page width.
  • 1 / 2 — half width.
  • Full width — edge to edge. Best for banners and hero shots.
